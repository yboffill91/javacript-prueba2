# Arreglos

## Declaracion

``` js
    const arreglo=["elem0","elem1","elemN"];
    let arreglo2=["elem0","elemN"];
    console.log(arreglo2[0]);
```

Una vez declarado el arreglo ocupa todas las posiciones, las que no estan declaradas las define como undefined.
